---
title: 'Delete Plant'

Delete a plant from the Vodex.ai Plant Store API by its ID. This endpoint demonstrates Vodex.ai's secure API documentation with proper parameter validation and response handling.

<ParamField path="id" type="integer" required>
The unique identifier of the plant to delete. Must be a valid integer.
</ParamField>

<ResponseExample>
```http Success Response
HTTP/1.1 204 No Content
```
</ResponseExample>
